```
変数$test_01、変数$test_02のどちらかがtrueの場合、「OR」の文字列を返す処理を記述してください。
```
------------------
```
<?php
function chapter03_02() {
    $test_01 = false;
    $test_02 = true;
    # === 以下に記載 ===
    if ($test_01 || $test_02){
    return 'OR';
    }
}

print chapter03_02()
?>
```
